Public Healthcare Data Repository
#########################################

-- Objective --
The project is intended to create interactive front end from which users can easily access all publicly available Medicare, Medicaid, Medi-Medi, State, and Regional healthcare data, including but not limited to beneficiary, providers, clearinghouses, contractors, data systems, codes (procedure, diagnosis, conditions, screen, etc), and claims, provide simple and modern visualization of the contents, query cross-source data, and share high quality insights and analytics.
