# Development Logs

This directory contains structured pre-development briefs and post-development retrospectives.

## Naming Convention

```
YYYY-MM-DD_<task-slug>_brief.md      — Pre-development planning brief
YYYY-MM-DD_<task-slug>_retro.md      — Post-development retrospective
```

## Purpose

- Build institutional memory across Claude sessions
- Track decision rationale so future sessions don't re-litigate settled decisions
- Surface recurring patterns, pitfalls, and effective approaches
- Provide context continuity when resuming after breaks
